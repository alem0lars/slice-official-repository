    Slice Official Repository

    Introduction

    This is a comprehensive repository for configuration files (a.k.a. dotfiles).

    It's made by perfectionists and people obsessed for elegant and perfect configurations.

    Usage

    At the moment you need to manually install/update your configuration files because there isn't a provision system to automatically update your system configuration, but it is in the plans.
